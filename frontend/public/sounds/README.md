# Sound Effects

This directory should contain the following sound effect files:

## Required Files

1. **chef-kiss.mp3** - Plays when Konami code is triggered
   - Suggested: A satisfying "mwah" kiss sound or chef's kiss effect
   - Duration: ~1 second

2. **success-chime.mp3** - Plays during confetti/celebration
   - Suggested: A pleasant success chime or achievement sound
   - Duration: ~1-2 seconds

3. **pop.mp3** - Plays when logo secret is revealed
   - Suggested: A soft pop or bubble sound
   - Duration: ~0.5 seconds

4. **ding.mp3** - Plays when secret word is discovered
   - Suggested: A notification ding or bell
   - Duration: ~0.5 seconds

## Free Sound Resources

You can find royalty-free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/
- https://pixabay.com/sound-effects/

## Note

Sound playback respects the user's mute preference stored in localStorage.
The app will gracefully handle missing sound files by catching playback errors.
